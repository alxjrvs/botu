QUICK START!

Install! one

# one binary onto your PATH
curl -fsSL https://raw.githubusercontent.com/alxjrvs/boom/main/install.sh | sh

macOS arm64/x64, Linux x64 — nothing else required. Homebrew works too; see the Guide.

Bootstrap! two

# clone your dotfiles repo and sync it
boom source set you/dotfiles   # your GitHub user/repo

A bare machine, claimed in one shot — clone plus a first boom source.

Stay in sync! three

boom source   # pull the repo, reconcile
boom verify   # hunt for drift

The repo is the source of truth. verify reports drift (exit 0 ok, 2 warn, 1 fail); source --fix mends it.

boom source skips conflicting files it doesn’t own by default — pass --fix to overwrite them.
